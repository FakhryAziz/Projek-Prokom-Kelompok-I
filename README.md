# SPBU App

Aplikasi Flask sederhana untuk memantau ketersediaan BBM di beberapa SPBU.

## Menjalankan lokal

```bash
python app.py
```

Akses di http://127.0.0.1:5000/login

## Deploy ke Railway

Project ini sudah disiapkan dengan:
- `requirements.txt`
- `Procfile`
- dukungan port dari environment variable `PORT`
